# Timer App

An timer app which counts down from a specified time towards 0

# Live Demo

To see the app in action, please visit [https://timer-js-mauve.vercel.app](https://timer-js-mauve.vercel.app)

# Overview

*Timer* is a simple timer application that counts down from a specified time towards 0

- Features:
	- Input: type any numbers of seconds you's like to be counted down to 0
	- Play: start the timer
	- Pause: pause the timer
	- Progress ring will decrease in a synchronic fashion with the timer
